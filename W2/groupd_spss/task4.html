Example

Check the normality of the video quality (VQ) ratings for condition 26 for game4 (using both Statistical Tests and Q-Q plot).

Dataset: Gaming_Video_Quality_dataset.xlsx

Setup and transfering data format

Simply import the data from the Excel file.

Performing the test

  1. Select Cases -> Check “If condition is satisfied” in the “Select” section
  2. Click on “If…”-button -> Define condition “Condition = 26 AND Game = "Game4"
  1. Analyze -> Descriptive Statstics -> Explore…
  2. Add the “VQ” variable in the “Dependent List”
  3. Make sure the “Both”-option is checked in the “Display” section
  4. Click on “Plots” -> Check the “Normality plots with tests” option
